# CONTRIBUTING

FEEL FREE TO CREATE ISSUES AND PRs

## Development

### How to start?

#### Requirements

- OCaml >= 4.14
- Dune
- OPAM

#### Use development branch

Use the `dev` branch.

#### Setup development environment

```sh
# Create a switch for project
$ opam switch create . --deps-only
# Or just install deps
$ opam install . --deps-only
```
