# CONTRIBUTING

FEEL FREE TO CREATE ISSUES AND PRs

## Development

### Requirements

- OCaml > 4.14
- Dune
- OPAM

```sh
# Create a switch for project
$ opam switch create . --deps-only
# Or just install deps
$ opam install . --deps-only
```

### How to start?

See [development demo](./tests/demo/).
